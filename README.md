# 득템시루 랜딩 페이지

마감 임박 할인 상품을 구매자와 판매자에게 연결하는 득템시루 프로젝트의 GitHub Pages용 포트폴리오 랜딩 페이지입니다.

## 로컬 확인

정적 HTML/CSS/JS로 구성되어 있어 별도 빌드 없이 `index.html`을 브라우저에서 열면 됩니다.

## GitHub Pages 배포

1. GitHub 저장소의 `Settings`로 이동합니다.
2. `Pages` 메뉴에서 `Deploy from a branch`를 선택합니다.
3. Branch를 `main`, folder를 `/ (root)`로 설정합니다.
4. 저장 후 발급되는 Pages URL을 프로젝트 README와 이력서에 연결합니다.
